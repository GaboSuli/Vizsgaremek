# Bevezetés
Ez a vizsgaremek arrol szol, hogy olyan felületet adjunk embereknek melyen könnyebben tudják dokumentálni, kigondolni, és átgondolni költségeiket.
Azért választottuk ezt a témát, mert sok kortársunk, illetve ember akit ismerunk nem tud bánni pénzével, illetve nem tudja felismerni értékét.
# DOKUMENTÁCIO
A fejlesztöi környezetek amelyeket használunk:
- Laravel
- React, melyben modulok:
-- axios
-- bootstrap
-- chart.js
-- cors
-- date-fns
-- express
-- react-bootstrap
-- react-chartjs-2

- Visual Studio Code
